3M Health Information Systems utiliza PNL para potenciar la codificación asistida por computadora (CAC) y la documentación clínica asistida por computadora (CDI). En lugar de que los analistas lean detenidamente la documentación clínica y produzcan códigos, pueden usar su experiencia para revisar los códigos sugeridos automáticamente por el sistema y convertir la documentación clínica en un amplio repositorio de datos.
Especificamente, el sistema CodeAssist℠ de 3M℠ CodeRyte℠ es un sistema basado en la web que aprovecha la PNL avanzada para examinar el texto completo del informe de un médico y asignar automáticamente códigos precisos de CPT® e ICD. Al identificar los códigos de facturación correctos y simplificar el proceso de flujo de trabajo de codificación, las organizaciones pueden mejorar el cumplimiento y la eficiencia al tiempo que reducen las denegaciones, los días A / R y el costo total a cobrar.
CodeAssist℠ de 3M℠ CodeRyte℠ , se basa  en un sistema híbrido ya que  combina un  clasificador de texto de Deep learning  y un sistema basado en reglas duras, que se utiliza para mejorar aún más los resultados. La ventaja de este tipo de sistemas híbridos es que se pueden ajustar fácilmente agregando reglas específicas para aquellas etiquetas en conflicto que el algoritmo de deep learning no pueda modelar correctamente.
El primer componente del algoritmo usa un conjunto de reglas lingüísticas hechas a mano. Estas reglas instruyen al sistema a utilizar elementos semánticamente relevantes de un texto para identificar categorías relevantes en función de su contenido. 
El segundo componente, usa ejemplos pre-etiquetados como datos de entrenamiento para un algoritmo de Deep Learning que puede aprender las diferentes asociaciones entre partes de texto y que se espera una salida particular (es decir, etiquetas) para una entrada particular (es decir, texto). Utiliza el algoritmo word2vec para realizar la extracción de características y un algortimo de Deep learning (redes neuronales) para la predicción.
